# @orchestra-mcp/widgets

Utility widget components for Orchestra MCP

Part of the [Orchestra MCP](https://orchestra-mcp.dev) ecosystem.

## Install

```bash
npm install @orchestra-mcp/widgets
# or
pnpm add @orchestra-mcp/widgets
```

## Usage

```tsx
import { } from '@orchestra-mcp/widgets'
```

## Requirements

- React >= 18.0.0
